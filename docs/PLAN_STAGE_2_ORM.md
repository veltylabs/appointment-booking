## Step 1 — model.go

Define all structs here with `db:` tags. Run `ormc` after to regenerate `model_orm.go`.

**Critical tag rules:**
- `db:"pk"` → primary key. **`ID` fields are auto-detected as PK** (`tinywasm/fmt.IDorPrimaryKey`) — only add this tag if your PK field has a different name.
- `db:"unique"` → UNIQUE constraint
- `db:"not_null"` → NOT NULL constraint
- `db:"-"` → exclude field from ORM (computed fields, slices, etc.)
- Composite tags: `db:"unique,not_null"`
- **`TableName()`** — **do NOT declare it**. `ormc` auto-generates it as snake_case of the struct name (`EmployeeServiceConfig` → `employee_service_config`, `Reservation` → `reservation`, etc.).

```go
//go:build !wasm

package appointmentbooking


// EmployeeServiceConfig maps per-staff-per-service durations and overrides.
type EmployeeServiceConfig struct {
    ID              string // auto-detected as PK by ormc
    TenantID        string
    StaffID         string
    ServiceID       string
    DurationMin     int64
    BufferMin       int64
    PriceOverride   float64
    PaymentRequired bool
    IsActive        bool
}

// WorkCalendarConfig is the single source of truth for timezone per staff.
type WorkCalendarConfig struct {
    ID       string // auto-detected as PK by ormc
    TenantID string
    StaffID  string
    Timezone string // IANA e.g. "America/Santiago"
    IsActive bool
}

// WorkCalendarWeekly defines recurring weekly hours for a staff member.
type WorkCalendarWeekly struct {
    ID          string // auto-detected as PK by ormc
    TenantID    string
    StaffID     string
    DayOfWeek   int64 // 0=Sunday … 6=Saturday
    WorkStart   int64 // minutes from midnight, local time
    WorkFinish  int64
    BreakStart  int64
    BreakFinish int64
    IsActive    bool
}

// WorkCalendarException overrides working hours for a specific date.
type WorkCalendarException struct {
    ID            string // auto-detected as PK by ormc
    TenantID      string
    StaffID       string
    SpecificDate  int64  // unix timestamp (UTC midnight)
    ExceptionType string // "day_off" | "custom_hours"
    StartTime     int64
    EndTime       int64
    Notes         string
}

// Reservation is the core booking record.
type Reservation struct {
    ID                      string // auto-detected as PK by ormc
    TenantID                string
    ClientID                string
    CreatorUserID           string
    EmployeeServiceConfigID string
    StaffIDSnapshot         string
    ServiceIDSnapshot       string
    DurationMinSnapshot     int64
    PriceSnapshot           float64
    CurrencySnapshot        string
    ReservationDate         int64  // unix timestamp of the LOCAL date (UTC midnight)
    ReservationTime         int64  // unix timestamp (UTC)
    LocalStringDate         string // "2026-03-04"
    LocalStringTime         string // "14:30"
    Status                  string // FSM state — use constants from fsm.go (StatusPending, StatusConfirmed, etc.)
    RescheduledFromID       string
    PaymentID               string
    Notes                   string
    UpdatedAt               int64
    UpdatedBy               string
    Revision                int64
}
```

After defining structs, run from the package directory:
```bash
ormc
```
This regenerates `model_orm.go` with `Schema()`, `Values()`, `Pointers()`, the `XxxMeta_` descriptor struct, and the typed `ReadOneXxx` / `ReadAllXxx` helpers.

## Step 2 — repository.go

> **Read this section fully before writing any code.**

### ORM API — Complete Reference

The ORM has TWO separate APIs. Never mix them.

#### A) Direct operations (`db.Create`, `db.Update`, `db.Delete`)

```go
// CREATE — always set string PK via unixid BEFORE calling Create
model.ID = unixid.Generate()
err := db.Create(&model)

// UPDATE — accepts optional conditions as variadic Condition args.
// orm.Eq, orm.Gt, orm.Lt, orm.Neq, orm.Gte, orm.Lte, orm.Like, orm.In are helpers.
// ALWAYS use the typed descriptor (Xxx_) — NEVER raw strings.
err := db.Update(&model)                                                    // update by PK (SQLite compiler default)
err := db.Update(&model, orm.Eq(Reservation_.Status, StatusPending))        // update with WHERE clause — use FSM constants, NEVER raw strings

// DELETE
err := db.Delete(&model, orm.Eq(Reservation_.ID, id))
```

#### B) Query builder (`db.Query(model).Where(...).ReadOne()` / `ReadAll(...)`)

```go
// The generated descriptor struct (XxxMeta_ or Xxx_) contains column name constants.
// Use these — NEVER use raw string column names.
//
// Example for EmployeeServiceConfig:
//   EmployeeServiceConfig_.ID        → "id"
//   EmployeeServiceConfig_.StaffID   → "staff_id"
//   EmployeeServiceConfig_.TenantID  → "tenant_id"

// READ ONE (single result)
model := &EmployeeServiceConfig{}
qb := db.Query(model).
    Where(EmployeeServiceConfig_.ID).Eq(id)
result, err := ReadOneEmployeeServiceConfig(qb, model)
if err != nil {
    return EmployeeServiceConfig{}, err  // includes orm.ErrNotFound when row absent
}

// READ ALL (multiple results)
proxy := &Reservation{}
qb := db.Query(proxy).
    Where(Reservation_.TenantID).Eq(tenantID).
    Where(Reservation_.StaffIDSnapshot).Eq(staffID).
    Where(Reservation_.ReservationDate).Gte(from).
    Where(Reservation_.ReservationDate).Lte(to)
rows, err := ReadAllReservation(qb)

// OR condition
qb := db.Query(proxy).
    Where(Reservation_.Status).Eq("pending").
    Or().Where(Reservation_.Status).Eq("confirmed")
```

#### C) Error handling

```go
import (
    "errors"
    "github.com/tinywasm/orm"
)

result, err := ReadOneWorkCalendarConfig(qb, model)
if errors.Is(err, orm.ErrNotFound) {
    return WorkCalendarConfig{}, ErrNotFound  // re-export as package sentinel
}
if err != nil {
    return WorkCalendarConfig{}, err
}
```

> **`orm.ErrNotFound`** is returned by `ReadOne()` when the SQL driver returns `sql.ErrNoRows`.
> Do NOT compare against `"sql: no rows in result set"` string — always use `errors.Is(err, orm.ErrNotFound)`.

#### D) Transactions

```go
err := db.Tx(func(tx *orm.DB) error {
    // All db.Create / db.Update / db.Query calls inside here use the same transaction.
    // Returning a non-nil error triggers automatic Rollback.
    // Returning nil triggers Commit.
    
    current := &Reservation{}
    qbTx := tx.Query(current).Where(Reservation_.ID).Eq(id)
    got, err := ReadOneReservation(qbTx, current)
    if err != nil {
        return err
    }
    if got.Revision != expectedRevision {
        return ErrConflict
    }
    got.Status    = newStatus
    got.UpdatedBy = updatedBy
    got.UpdatedAt = updatedAt
    got.Revision++
    return tx.Update(got)
})
```

```go
// UpdateReservationStatus uses optimistic concurrency via revision check inside a DB Transaction.
func (r *Repository) UpdateReservationStatus(id, status, updatedBy string, updatedAt int64, expectedRevision int64) error {
    return r.db.Tx(func(tx *orm.DB) error {
        current := &Reservation{}
        qb := tx.Query(current).Where(Reservation_.ID).Eq(id)
        got, err := ReadOneReservation(qb, current)
        if errors.Is(err, orm.ErrNotFound) {
            return ErrNotFound
        }
        if err != nil {
            return err
        }
        if got.Revision != expectedRevision {
            return ErrConflict
        }
        got.Status    = status
        got.UpdatedBy = updatedBy
        got.UpdatedAt = updatedAt
        got.Revision++
        return tx.Update(got)
    })
```

#### workcalendar_config (Upsert pattern)

> `tinywasm/orm` has no native `Upsert`. Use **try-Create then Update on conflict**:

```go
func (r *Repository) UpsertCalendarConfig(cfg WorkCalendarConfig) error {
    // Try to find existing record for this (tenantID, staffID)
    existing := &WorkCalendarConfig{}
    qb := r.db.Query(existing).
        Where(WorkCalendarConfig_.TenantID).Eq(cfg.TenantID).
        Where(WorkCalendarConfig_.StaffID).Eq(cfg.StaffID)
    got, err := ReadOneWorkCalendarConfig(qb, existing)
    if err != nil && !errors.Is(err, orm.ErrNotFound) {
        return err
    }
    if errors.Is(err, orm.ErrNotFound) {
        // Does not exist — create
        cfg.ID = unixid.Generate()
        return r.db.Create(&cfg)
    }
    // Exists — update in place (preserve original ID)
    cfg.ID = got.ID
    return r.db.Update(&cfg)
}

func (r *Repository) GetCalendarConfig(tenantID, staffID string) (WorkCalendarConfig, error) {
    m := &WorkCalendarConfig{}

    if errors.Is(err, orm.ErrNotFound) {
        return WorkCalendarConfig{}, ErrNotFound

        return WorkCalendarConfig{}, err

}
```

### Repository struct

```go
//go:build !wasm

package appointmentbooking

import (
    "errors"

    "github.com/tinywasm/fmt"
    "github.com/tinywasm/orm"
    "github.com/tinywasm/unixid"
)

var (
    ErrNotFound = fmt.Err("record", "not", "found")
    ErrConflict = fmt.Err("optimistic", "concurrency", "conflict")
)

// Repository provides CRUD operations for all appointment-booking tables.
type Repository struct {
    db *orm.DB
}

// NewRepository creates a new Repository and auto-migrates all tables.
func NewRepository(db *orm.DB) (*Repository, error) {
    r := &Repository{db: db}
    tables := []orm.Model{
        &EmployeeServiceConfig{},
        &WorkCalendarConfig{},
        &WorkCalendarWeekly{},
        &WorkCalendarException{},
        &Reservation{},
    }
    for _, t := range tables {
        if err := db.CreateTable(t); err != nil {
            return nil, err
        }
    }
    return r, nil
}
```

```go
//go:build !wasm

package appointmentbooking

import (
    "errors"
    "testing"

    "github.com/tinywasm/sqlite"
)

func newTestRepo(t *testing.T) *Repository {
    t.Helper()
    db, err := sqlite.Open(":memory:")
    if err != nil {
        t.Fatalf("sqlite.Open: %v", err)
    }
    t.Cleanup(func() { db.Close() })
    repo, err := NewRepository(db)
    if err != nil {
        t.Fatalf("NewRepository: %v", err)
    }
    return repo
}
```
